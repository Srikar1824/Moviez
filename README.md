# Moviez 

##  Overview

Moviez is a web application that allows users to discover and explore movies using data from a public API. It provides a simple and interactive interface where users can search, filter, and sort movies.

---

##  Purpose

The purpose of this project is to build a web application using JavaScript that integrates a public API and demonstrates concepts like asynchronous data handling and dynamic UI updates.

---

##  API Used

This project uses the TMDB (The Movie Database) API to fetch movie data such as titles, posters, ratings, and release dates.

---

##  Features

* Fetch and display movies dynamically
* Search movies by title
* Filter movies based on rating
* Sort movies by rating and release date
* Responsive user interface

---

##  Tech Stack

* HTML
* CSS
* JavaScript
* Fetch API

---

##  How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser
